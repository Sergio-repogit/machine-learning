# Unsupervised Learning

Subrepositorio dedicado a técnicas de **aprendizaje no supervisado** para exploración de estructura latente y reducción de dimensionalidad.

Los proyectos se centran en segmentación de datos y transformación de espacios de alta dimensión.

---

## Contenido

### Clustering

Proyectos de agrupamiento sobre distintos datasets.

**Modelos utilizados:**

* K-means
* Gaussian Mixture Models (GMM)
* DBSCAN
* Clustering jerárquico
* Spectral clustering

**Objetivo:**
Identificar patrones y estructuras ocultas en los datos.

---

### Dimensionality Reduction

Proyectos orientados a la reducción de dimensionalidad.

**Técnicas utilizadas:**

* Principal Component Analysis (PCA)

**Objetivo:**
Reducir la dimensión de los datos preservando la máxima información posible y facilitando la visualización.

---

## Tecnologías

* R
* cluster

## Uso

Abrir el script correspondiente en RStudio y ejecutar el análisis deseado.
