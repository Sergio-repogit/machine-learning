# Supervised Learning

Subrepositorio dedicado a modelos de **aprendizaje supervisado** para tareas de regresión y clasificación, implementados principalmente en R.

Los proyectos incluidos siguen el flujo habitual de trabajo: preprocesamiento de datos, entrenamiento del modelo, validación y análisis de resultados.

---

## Contenido

### Regression

Proyectos orientados a la predicción de variables continuas.

**Modelos utilizados:**

* Regresión lineal
* Regresión polinómica
* Splines
* Step functions

**Datasets principales:**

* Datos económicos
* Dataset USA

---

### Classification

Proyectos de clasificación sobre distintos conjuntos de datos.

**Modelos utilizados:**

* Regresión logística
* Árboles de decisión
* Random Forest
* Support Vector Machines (SVM)
* K-Nearest Neighbors (KNN)

**Datasets principales:**

* Breast Cancer
* Iris
* Dataset Default

---

## Tecnologías

* R
* caret
* randomForest
* e1071
* tidyverse

---

## Uso

Abrir el script correspondiente en RStudio y ejecutar las celdas o bloques del proyecto deseado.
